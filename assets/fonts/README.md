# Polices requises

Téléchargez les polices suivantes:

## Noto Kufi Arabic
- URL: https://fonts.google.com/noto/specimen/Noto+Kufi+Arabic
- Fichiers requis:
  - NotoKufiArabic-Regular.ttf
  - NotoKufiArabic-Bold.ttf

## Roboto (optionnel, inclus dans Flutter)
- URL: https://fonts.google.com/specimen/Roboto
- Fichiers requis:
  - Roboto-Regular.ttf
  - Roboto-Bold.ttf

Placez tous les fichiers .ttf dans ce dossier assets/fonts/
